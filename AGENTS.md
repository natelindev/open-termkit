# Agent Instructions

- Before changing UI, read `design.md` and follow the Open Termkit design system.
- Use the Vercel-inspired Geist style: near-white or near-black canvas, precise hairline cards, restrained monochrome chrome, and tight sans typography.
- Keep dark theme support first-class. Any new UI must work in both light and dark themes through CSS variables.
- Keep the theme switch visible in app chrome and preserve its local-storage behavior.
- Do not return to the old cream monospaced ASCII-marker style.
